# commit_explainer

Weekly plain-English changelog generated by Claude. Collects all commits from the last 7 days and summarises what changed and why — grouped by theme, written for humans, not machines.

## How it works

1. Walks `git log HEAD` for commits in the last 7 days (capped at 100)
2. Builds a compact summary: date, short SHA, subject line, changed files
3. Truncates to 8,000 characters if the week was unusually active
4. Sends to Claude (`claude-haiku-4-5-20251001`) with instructions to group changes by theme and write in past tense
5. Returns the changelog as the run message — delivered via email

## Output format

```
## Weekly Changelog — 2026-03-07 to 2026-03-14

### Bot improvements
- Added HTML email templates with markdown rendering for cleaner notifications.
- Fixed a YAML boolean parsing bug that silently suppressed all email notifications.

### Infrastructure
- Added framework.log sink to capture scheduler and notifier output persistently.
```

## Required env vars

```
ANTHROPIC_API_KEY=
EMAIL_SENDER=
EMAIL_PASSWORD=
EMAIL_RECEIVER=
```

## Schedule

Every Monday at 07:00 UTC (`0 7 * * 1`) — recap of the previous week lands at the start of your work day.
